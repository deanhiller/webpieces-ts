# microsvc

Anchor directory for the virtual Nx `microsvc` project created by
`@webpieces/nx-webpieces-rules`. It hosts the runtime microservice-graph
targets (e.g. `nx run microsvc:visualize`).

Services are projects whose root matches `runtime-architecture.servicePaths` in
`webpieces.config.json`; each must declare a `service-contract.json`
(the api projects it implements + uses).

The committed runtime graph itself lives next to the architecture graph at
`architecture/runtime-dependencies.json` (generated by `nx run architecture:generate`).
