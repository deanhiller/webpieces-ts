Legend — fill = framework (libType), border = role

angular: Angular front-end
react: React front-end
browser: browser front-end base env
express: Express / server side
node: node server base env
server: runnable server app (thick green border)
client: client app, e.g. angular (thick red border)
designed-lib: library with a generated @DocumentDesign design (bold border)
lib: plain library, no generated design (thin border)
Each node label shows its dependency level (L#), its framework env set (e.g. [browser, node]), and its role. Rows are laid out by level (top = no dependencies), with the deepest libraries at the bottom. Transitive dependencies are allowed but not shown.
